# SODV1201S26Group1Project

Group 1 final project repository for SODV1201 Web Programming, Spring 2026 at Bow Valley College.

## Project Overview

This project is a shared workspace rental web application. Owners can register, log in, add properties, and list available workspaces. Coworkers can register, log in, search for available workspaces, view workspace details, and access the owner's contact information.

## Main Roles

- Owner: adds and manages properties and workspaces.
- Coworker: searches for available workspaces and views workspace details.
